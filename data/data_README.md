Coloca aquí el dataset con la estructura:

```
data/
├── train/
│   ├── american_football/*.jpg
│   ├── baseball/*.jpg
│   └── ... (15 clases)
└── test/

    └── ...
```

no se sube a GitHub por su tamaño). Descárgalo desde:
