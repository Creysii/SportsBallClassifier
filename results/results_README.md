# Carpeta `results/`

Aquí se guardan automáticamente las salidas de `python main.py`:

```
results/
├── features/   # features_train.json, features_test.json
└── figures/    # matrices de confusión (.png/.csv) y model_comparison (.png/.csv)
```

Se deja vacía intencionalmente en el repositorio; se genera al ejecutar el pipeline.
